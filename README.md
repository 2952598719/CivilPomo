# CivilPomo

一个把番茄钟和科技树结合起来的学习工具。

每个番茄钟完成会推进文明科技树的研发进度，从原始时代一路发展到信息时代。完成的节点会生成一段叙事文本，记录这段"文明史"。

## 功能

- 番茄钟计时（工作/短休息/长休息）
- 9 个时代、152 个节点的科技树
- 节点按科技/人文/民政分类，有前驱依赖关系
- 完成节点后 AI 生成叙事文本
- 编年史页面回顾文明历程
- 暗色/亮色主题切换
- 邮箱验证码登录，单设备互斥

## 技术栈

Next.js (App Router) + React + TypeScript + Tailwind CSS + Zustand

数据存储：Upstash Redis
部署：Vercel
